## Steps to set up you local enviroment

- **Prerequisites:**

- **NodeJS**: [Donwload NodeJS](https://nodejs.org/en/)

1. **Clone repository:**

```sh
git clone git@github.com:stefanoarce/auto-complete.git
```

2. **Install Node Packages**

```sh
npm i
```

2. **Start the project**

```sh
npm run start
```
